Truthiness

Truthiness is not so difficult to understand if you start with just primitive types. Just ask this simple question:

  • does the value cast to the Boolean value true or false?

    • If the value casts to true, it's truthy.
    If the value casts to false, it's falsy.

    To discover a value's truthiness you can either memorize a whole bunch of rules and make lots of guesses, or ... ask JavaScript! (just try it your debugger ;)
